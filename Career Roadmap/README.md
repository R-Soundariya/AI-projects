# 🧭 Data/AI Analyst — 90-Day Flight Plan

A personal career-switch roadmap: Operations/Support → Data/AI Analyst, built around 12–20 hrs/week and free resources only.

**🔗 Live version:** [https://r-soundariya.github.io/AI-projects/Career%20Roadmap/](https://r-soundariya.github.io/AI-projects/Career%20Roadmap/)

## What's in it

- **Skill gap analysis** — every skill the target role needs, ranked by how often it actually gates an interview
- **12-week plan** — week-by-week themes with specific free resources (SQLBolt, Kaggle Learn, StatQuest, Microsoft Learn, DeepLearning.AI)
- **3 portfolio projects** — each with a ready-to-adapt resume line
- **Timeline & milestones** — including an honest read on how long the job search itself takes after the 90 days

## How to run

No install, no server — download `index.html` and double-click it. Includes a light/dark toggle that remembers your choice.
